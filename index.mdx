wx-cli is a command-line tool that lets you query your local WeChat data directly from the terminal. It decrypts WeChat's SQLCipher-encrypted databases on-the-fly using a background daemon, so your data never leaves your machine. Output defaults to YAML for easy reading and AI integration, with `--json` available for piping to `jq` and other tools.

## Key features

- **Zero-dependency binary** — single Rust binary, install with one command
- **Millisecond responses** — daemon keeps decrypted databases cached; mtime-aware so it only re-decrypts when files change
- **Fully local** — data never leaves your machine; real-time on-demand decryption
- **AI-friendly output** — YAML by default to save tokens; switch to JSON with `--json`
- **Cross-platform** — macOS (Apple Silicon + Intel), Linux (x86\_64 + arm64), Windows

## Get started

<Steps>
  <Step title="Install wx-cli">
    ```bash
    npm install -g @jackwener/wx-cli
    ```
  </Step>
  <Step title="Initialize (run once)">
    Keep WeChat running, then:
    ```bash
    sudo wx init
    ```
    wx-cli scans WeChat's process memory to extract the database encryption key and saves your configuration to `~/.wx-cli/`.
  </Step>
  <Step title="Query your data">
    ```bash
    wx sessions          # recent conversations
    wx history "Alice"   # chat history with Alice
    wx search "meeting"  # search all messages
    ```
  </Step>
</Steps>

<Note>
  wx-cli only accesses **your own** WeChat data stored locally on your device. It does not connect to WeChat servers or transmit any data externally.
</Note>
